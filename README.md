# Tumor-Growth-Modeling
Tumor Growth Modeling in R

This project analyzes longitudinal tumor growth data from placebo-treated mice.

Methods

Data cleaning performed in R

Nonlinear least squares regression using:

Exponential model

Logistic model

Gompertz model

Parameters estimated per mouse independently

Structure

/data – cleaned datasets

/scripts – R scripts for preprocessing and model fitting
